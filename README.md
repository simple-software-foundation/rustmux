# rustmux architecture

```
                                                    MUX
                                    |---------------------------------|

========        ============        =======                     =======        ========
|      | stdin  |          | stdin  |     | stdin   ...  stdin  |     | stdin  |      |
|      + =====> +          | =====> +     | =====>       =====> +     | =====> +      |
| USER |        | TERMINAL |        | PTY |         ...         | PTY |        | PROC |
|      + <===== |          + <===== |     + <=====       <===== |     + <===== +      |
|      | stdout |          | stdout |     | stdout  ...  stdout |     | stdout |      |
========        ============        =======                     =======        ========
```
